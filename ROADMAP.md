# Roadmap

## Phase 1: Environment & Scaffolding
- DevContainer, scripts, baseline API project, React skeleton

## Phase 2: Domain Foundations
- Entities: Game, Table, Customer, Reservation, Order, MenuItem, Event, GameSession

## Phase 3: Core REST Endpoints
- CRUD + action endpoints (checkout, return, availability, recommendations)

## Phase 4: Testing Infrastructure
- Unit + Integration + Playwright setup, data builders

## Phase 5: Bug Demonstrations
- Branches: midnight-reservation, double-discount, vanishing-game, table-time-traveler, order-item-duplication, case-sensitive-email, event-registration-race, loyalty-points-reversal

## Phase 6: CI/CD & Polishing
- GitHub Actions, coverage, artifacts, documentation refinement

Issues will link each milestone to concrete acceptance criteria.
